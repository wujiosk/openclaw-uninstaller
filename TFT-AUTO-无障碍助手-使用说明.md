# TFT 无障碍自动识别助手

## 当前能力

- 自动抓取当前屏幕截图
- 自动识别等级、金币、血量
- 自动 OCR 商店文本
- 自动生成买卡节奏建议
- 大字显示当前主建议
- 语音播报建议变化

## 启动

在 `E:\claw` 目录执行：

```bash
npm run start:webui
```

然后打开：

```text
http://127.0.0.1:32123/tft-auto.html
```

## 使用步骤

1. 打开云顶之弈对局画面
2. 打开上面的本地页面
3. 点击“启动自动识别”
4. 程序会自动截图并识别
5. 大字建议和语音播报会自动更新

## 识别不到时怎么调

如果金币、等级、血量或商店识别不准：

1. 展开“区域预设与微调”
2. 调整 `x / y / w / h`
3. 再点击“手动识别一次”

这些数值是相对屏幕的百分比：

- `x`：横向起点
- `y`：纵向起点
- `w`：宽度
- `h`：高度

## 说明

- 这是本地无障碍辅助，不自动点击游戏
- 不接管鼠标和键盘
- 默认按 16:9 全屏客户端预设
- 不同分辨率、窗口模式、缩放比例下，通常需要微调区域
- 商店中文 OCR 会受字体、分辨率和画面特效影响，第一版更适合做辅助判断，不适合作为唯一依据
