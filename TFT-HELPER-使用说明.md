# 云顶之弈阵容与装备助手

## 功能

- 根据当前等级、金币、血量和阶段，给出经济节奏建议
- 根据已有棋子和商店来牌，给出阵容方向和买卡优先级
- 根据散件，给出可合成装备建议
- 本地运行，不控制游戏客户端，不自动点击

## 启动

在 `E:\claw` 目录执行：

```bash
npm run start:webui
```

然后打开：

```text
http://127.0.0.1:32123/tft-helper.html
```

## 输入说明

- `当前等级`：你现在的人口
- `当前金币`：本回合可用金币
- `当前血量`：当前剩余生命值
- `当前阶段`：例如 `3-2`、`4-1`
- `已有散件`：例如 `大剑 反曲弓 拳套`
- `当前商店来牌`：把这一轮商店 5 张牌输入进去
- `场上和备战席已有棋子`：把你现在已有的核心棋子输入进去
